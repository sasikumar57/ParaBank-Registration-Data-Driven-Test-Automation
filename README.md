## Project Overview
Automated the user registration workflow of a banking web application using **Selenium WebDriver** and **TestNG** with a data-driven approach.

## Key Highlights
- Validated multiple input datasets dynamically using **TestNG DataProvider** and **Apache POI (Excel)**
- Handled dynamic web elements using **implicit waits** and **JavaScriptExecutor**
- Covered **positive, negative, and duplicate registration scenarios** with reusable test scripts

## Tech Stack
Selenium WebDriver | TestNG | Java | Apache POI
